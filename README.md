<h2>📌 Project Overview</h2>

<p>
This project is a <b>modern, interactive Personal Portfolio Website</b> designed to showcase my complete professional journey, including my <b>projects, blogs, freelancing work, certifications, and extra-curricular activities</b>.
</p>

<p>
Unlike traditional static portfolios, this application provides a <b>dynamic Single Page Application (SPA) experience</b>, ensuring smooth navigation without page reloads and delivering a fast, seamless user experience.
</p>

<p>
The main goal of this project is to act as a <b>centralized platform</b> where recruiters, clients, and peers can easily explore my skills, work, and achievements in a structured and visually engaging way.
</p>

<p>
It serves as:
</p>

<ul>
  <li>💼 A <b>digital resume</b> representing my profile</li>
  <li>📂 A <b>project showcase platform</b> with detailed demonstrations</li>
  <li>🤝 A <b>client interaction system</b> via contact form</li>
  <li>⚙️ A <b>real-world full-stack application</b> demonstrating my technical skills</li>
</ul>

<hr>

<h2>📂 Portfolio Section (Core Feature)</h2>

<p>
The portfolio section is one of the most important parts of this project. It is designed to organize and present all my work in a clean and user-friendly way.
</p>

<p>
It includes the following categories:
</p>

<ul>
  <li>💻 Projects</li>
  <li>📝 Blogs</li>
  <li>💼 Freelancing Work</li>
  <li>🏆 Certifications</li>
  <li>🎯 Extra-Curricular Activities</li>
</ul>

<p>
On the main <b>index.html</b> page, each category is represented using images inside the portfolio section.
</p>

<p>
When a user clicks on any image, it opens a <b>separate dedicated page</b> where detailed information about that category is displayed.
</p>

<p>
This design ensures:
</p>

<ul>
  <li>✔️ Clean and uncluttered homepage</li>
  <li>✔️ Easy navigation and better user experience</li>
  <li>✔️ Detailed exploration of each section</li>
</ul>

<hr>

<h2>⚙️ How It Works</h2>

<ul>
  <li>📌 All sections are loaded inside a single HTML file</li>
  <li>📌 JavaScript dynamically switches sections (SPA routing)</li>
  <li>📌 Theme switcher updates UI styles in real-time</li>
  <li>📌 User interactions are handled without page reload</li>
  <li>📌 Contact form sends data using fetch API</li>
  <li>📌 Backend processes and stores data in SQLite database</li>
</ul>

<hr>

<h2>🌐 Backend API</h2>

<ul>
  <li><b>Endpoint:</b> <code>http://localhost:3000/api/contact</code></li>
  <li><b>Method:</b> POST</li>
  <li><b>Function:</b> Stores user messages from contact form</li>

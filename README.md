<h2>📌 Project Overview</h2>

<p>
This project is a <b>modern, interactive Personal Portfolio Website</b> designed to showcase skills, projects, and professional achievements in <b>RPA, Web Development, and AI-powered automation</b>.
</p>

<p>
Unlike traditional static portfolios, this application delivers a <b>dynamic Single Page Application (SPA) experience</b> with smooth navigation, real-time interactivity, and backend integration.
</p>

<p>
It serves as:
</p>

<ul>
  <li>💼 A <b>digital resume</b></li>
  <li>📂 A <b>project showcase platform</b></li>
  <li>🤝 A <b>client interaction system</b></li>
  <li>⚙️ A <b>demonstration of full-stack development skills</b></li>
</ul>

<hr>

<h2>🔥 Key Features</h2>

<ul>
  <li>⚡ Single Page Application (SPA) – No page reloads</li>
  <li>🎨 Dynamic Theme Switcher (Dark/Light + Color Skins)</li>
  <li>⌨️ Typing Animation using Typed.js</li>
  <li>📊 Interactive Skill Progress Bars</li>
  <li>📱 Fully Responsive Design</li>
  <li>🎥 Embedded Video Project Demonstrations</li>
  <li>📩 Full-Stack Contact Form with Database Storage</li>
</ul>

<hr>

<h2>⚙️ How It Works</h2>

<ul>
  <li>📌 All sections are loaded in a single HTML file</li>
  <li>📌 JavaScript dynamically switches sections (SPA behavior)</li>
  <li>📌 Theme switcher updates styles in real-time</li>
  <li>📌 Contact form sends data via API to backend</li>
  <li>📌 Backend stores user data in SQLite database</li>
</ul>

<hr>

<h2>🌐 Backend API</h2>

<ul>
  <li><b>Endpoint:</b> <code>http://localhost:3000/api/contact</code></li>
  <li><b>Method:</b> POST</li>
  <li><b>Function:</b> Stores contact form data in database</li>
</ul>

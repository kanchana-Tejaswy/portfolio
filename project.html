Password Generator

The Password Generator is a web-based application developed using HTML, CSS, and JavaScript that allows users to generate secure and customizable passwords instantly. The application enables users to control the length and complexity of the password through an interactive interface.

Users can include or exclude different character types such as uppercase letters, lowercase letters, numbers, and special symbols to create strong and secure passwords. The tool also provides a convenient copy-to-clipboard feature that allows users to quickly copy the generated password for use in other applications.

Key Features

  • Dynamic password length control using a slider
  • Option to include uppercase, lowercase, numbers, and symbols
  • Instant password generation using JavaScript
  • Copy password to clipboard functionality
  • Clean and responsive user interface

Technologies Used

  • HTML5 for structure
  • CSS3 for styling and layout
  • JavaScript for password generation logic and interactivity

This project helped strengthen my understanding of DOM manipulation, event handling, and building interactive web applications using core front-end technologies.
